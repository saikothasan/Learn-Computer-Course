Edit Shape

PowerPoint

  1. Click the shape that you want to change.

    To select multiple shapes, press and hold CTRL while you click the shapes.

    For more information about selecting shapes.

  2. Under Drawing Tools, on the Format tab, in the Insert Shapes group, click Edit Shape  .

    If you do not see the Drawing Tools or Format tabs, make sure that you selected a shape.

  3. Point to Change Shape, and then click the shape that you want.
